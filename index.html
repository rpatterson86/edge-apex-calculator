About This App

EDGE Apex Companion is a development build for calculating Grip Length and Support Clearance.

Current version: 1.7

  • ✅ App icon finalized
  • ✅ Hero image finalized
  • ✅ Grip Length calculator

  • ✅ Add to Home Screen support
  • ✅ Offline support
Install on iPhone

Open in Safari, tap Share, then choose Add to Home Screen.

Update Available

A newer version of EDGE Apex Companion is available.

  1. Open this app while connected to the internet.
  2. Tap Update Now.
  3. Close the Home Screen app completely.
  4. Reopen it.

If it still does not update, remove it from the Home Screen and add it again from Safari.
